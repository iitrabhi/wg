### Markdown Cheat Sheet

#### Basic Syntax:

- **Headings**:
    - `# H1`, `## H2`, `### H3`, etc.
- **Bold**: `**bold text**`
- **Italic**: `*italicized text*`
- **Bulleted List**:
    - `* item 1`
    - `* item 2`
- **Numbered List**:
    1. `1. First item`
    2. `2. Second item`
- **Links**: `[title](https://)`
- **Images**: `![alt text](image.jpg)`
- **Code Blocks**: ` ```code``` `
- **Inline Code**: `` `code` ``
- **Blockquotes**: `> blockquote`

#### Advanced Syntax:
- **Task List**:
    - `- [x] Write the press release`
    - `- [ ] Update the website`
